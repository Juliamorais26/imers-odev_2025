#Site Estante Ascendente

O Estante Ascendente é uma simulação de loja virtual de livros desenvolvida para apresentar, de forma leve e intuitiva, como seria a experiência em um e-commerce literário moderno.

Além de exibir títulos, preços e detalhes dos livros, o site foi criado com a intenção de inspirar o visitante a ter prazer pela leitura. Cada card, animação e detalhe visual foi pensado para tornar a navegação agradável e despertar curiosidade sobre novas histórias.

Um projeto simples, mas feito com cuidado,combinando design, funcionalidade e amor por livros.
